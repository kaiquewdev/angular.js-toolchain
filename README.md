Angular.js Tool Chain
=====================

A simple collection of tools to assist you 
in building applications with angular.js

Installation
============

    - npm install angular-toolchain

Usage
=====

Manual
======
    
    angular-toolchain:

        - man angular-toolchain

    angular.js:
        
        - man angular.js
